	file: 0-memset.c
Task1, memcpy:

	file: memcpy.c
Task2, strchr:
	write a function that locates a charcter in a string.

Task3, strspn:
	write a function that gets the length of a prefix substring.
	file: 3-strspn.c
Task4, strbrk:

	file: 4-strbrk.c
Task5, strstr:
